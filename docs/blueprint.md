# **App Name**: Action Board

## Core Features:

- Checklist Display: Display four items, two with checkboxes and two with play buttons.
- Checkbox Interaction: Allow users to check/uncheck items with checkboxes. If all items are completed, then a message appears that says the job is complete.
- Play Button Action: Make the play buttons trigger specific actions or navigation.
- Dynamic Tool Assignment: The play button incorporates a generative AI tool to analyze tasks and determine when to execute certain automated steps in an order
- Completion Feedback: When one item is checked off, ie, play button or checkbox clicked, make the play or check green in color and give the item a green border. when all items are checked off, play an animation for confetti.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) to represent calmness and focus.
- Background color: #f6571e
- Item background color: white
- Accent color: Soft teal (#66CDAA), offering a complementary but gentle contrast.
- Font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look for headlines and body text.
- Use clear, minimalist icons for checkboxes and play buttons. Style icons for the AI integration to subtly suggest a futuristic or smart aesthetic.
- Use a simple, single-column layout for clear task visibility.
- Subtle animations when tasks are checked or actions triggered; Confetti animation when all tasks are complete.
